List of git commands

Assuming the local repo is clean and updated with the the remote master repo:

  • Create a new branch where the changes will be made
    • $ git branch "temporary_branch" "temporary_branch"
  • Change to the new repo:
    • $ git checkout 'temporary_branch'

    At this point you can work on the files inside the directory. Example make changes or addition to your code

  • Check the status of the changes
    • $ git status
  • Add the changes to the stage area
    • $ git add -A
  • Commit the changes
    • $ git commit -m "Explain the differences"
  • Push the new local repo to the remote repo
    • $ git push -u origin subtract

    After all the tests are made and the new code is good to be used

  • Switch to the master repo
    • $ git checkout master
  • Pull the changes from the master repo
    • $ git pull origin master
  • Merge the 'temporary_branch' with the master repo
    • $ git merge 'temporary_branch'
  • Push the changes to the master
    • $ git push origin master

How to use grep recursively

You can search recursively i.e. read all files under each directory for a string “192.168.1.5”
$ grep -r "CSRF_COOKIE_SECURE" /~
"OR"
$ grep -R "CSRF_COOKIE_SECURE" ~/
